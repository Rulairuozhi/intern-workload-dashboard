四、使用指南（3 步启动）
============================================================================

第一步：安装依赖
----------------
打开 PowerShell/Terminal，进入项目目录：

  cd  e:\实习生工作时长可视化
  pip install -r requirements.txt

首次安装可能需要 2-3 分钟。

第二步：启动应用
----------------
  streamlit run App.py

应用会自动打开浏览器，显示 http://localhost:8501

第三步：使用应用
----------------
1. 点击侧边栏 Upload Excel File 上传数据
2. 确保 Excel 包含这些列：
   - Week (数字：1, 2, 3...)
   - Date (文本：日期范围)
   - 部门列 (PPM1, DP, PPM2, STA, PEH, IP1, IP2)
3. 选择要分析的部门和周数
4. 查看图表和洞察
5. 下载数据（可选）

八、可视化图表说明（非技术人员版本）
============================================================================

1. 周趋势折线图（第一行左）
   - 用途：看各部门工作量是否在增加、减少或稳定
   - 红线：32小时警告线
   - 上升趋势：警惕人力不足
   - 下降趋势：工作负荷释放

2. 部门对比柱状图（第一行右）
   - 用途：看哪些部门工作最多
   - 最高的柱子：需要关注资源
   - 用于：制定资源分配计划

3. 热力图（第二行）
   - 用途：快速看哪个部门在哪周最忙
   - 深色区域：高强度工作
   - 浅色区域：正常工作
   - 用于：发现规律和瓶颈

4. 忙碌周频率图（第三行左）
   - 用途：看哪个部门经常超时
   - 高的柱子：长期压力
   - 用于：评估工作强度

5. 工作量分布箱线图（第三行右）
   - 用途：看工作量是否稳定
   - 长的盒子：工作量不稳定
   - 中线位置高：长期忙碌
   - 用于：评估风险

九、故障排除
============================================================================

问题 1：找不到 Python
解决：确认已安装 Python 3.8+，cmd 输入 python --version

问题 2：pip 安装很慢
解决：使用清华源
  pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

问题 3：Excel 文件无法读取
解决：
- 确保是 .xlsx 或 .xls 格式
- 检查列名是否完全匹配
- 确保没有合并单元格

问题 4：Streamlit 无法启动
解决：
  pip install streamlit --upgrade
  streamlit run App.py --logger.level=debug

问题 5：云部署后没有数据保存
解决：这是正常的。Streamlit Cloud 不持久化会话数据
